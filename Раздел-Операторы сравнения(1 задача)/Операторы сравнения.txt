Каким будет результат этих выражений?

5>4
"ананас" > "яблоко"
"2" > "12"
undefined == null
undefined === null
null == "\n0n\"
null === +\"n0\n"

Решение:
5 > 4 - true
"ананас" > "яблоко" - Смотрим на первые буквы а(1) и я(32),1<32. Значит false
"2" > "12" - Сравнием первые цифры 2 и 1, 2>1. Значит true
undefined == null - Значения null и undefined равны только друг другу при нестрогом сравнении. Значит true
undefined === null - Строгое сравнение разных типов. Значит false
null == "\n0n\" - null равен только undefined. Значит false
null === +\"n0\n" - Строгое сравнение разных типов. Значит false
